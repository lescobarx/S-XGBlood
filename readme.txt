Full Execution Instructions

1. Obtain the Data Sets:

   * data_le.csv
   * data_le_val.csv
   * data_gw.csv
   * data_mi.csv

   The data sets from the University Medical Center Leipzig (UML) and University Medical Center Greifswald (UMG), as well as a conversion function for the MIMIC-IV data, are provided as an R package:
   https://github.com/ampel-leipzig/sbcdata (DOI: 10.5281/zenodo.6922968).

   Reference:
   Johnson A, Bulgarelli L, Pollard T, Horng S, Celi LA, Mark R. MIMIC-IV (version 1.0). PhysioNet. 2021. https://doi.org/10.13026/s6n6-xd98 

2. Run the Training Scripts:

   * training_CBC.py
   * training_CBC_PCT.py

3. Run the Seed Analysis:

   * seed_analysis.py
